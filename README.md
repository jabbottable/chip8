# chip8

chip8 is a emulator for the [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) programming language. The project is still in development.

## Installation

```bash
go build
```

## Usage

In a 64 character width terminal run:

```bash
./chip8 -f [file]
```
